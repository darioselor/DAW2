## 1. Utiliza SASS para crear variables CSS que representen diferentes tonalidades de colores (muy claro, claro, base, oscuro, muy oscuro) para los colores azul, morado y verde.

## 2. Genera clases globales que aplicarán estos colores como propiedades de texto en elementos HTML.

## 3. Las variables y clases deben seguir este formato (la opción por defecto no lleva guión):
`--color-nombreColor-tonalidad: codigoColor;`  
`.g-color-nombreColor-tonalidad { color: codigoColor; }`
